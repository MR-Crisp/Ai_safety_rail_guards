# AI Safety Rail Guards

This project demonstrates **AI safety guardrails** — mechanisms that sit between users and Large Language Models (LLMs) to **monitor, validate, and constrain AI behaviour**.

The system inspects prompts and model outputs to detect unsafe, non-compliant, or undesired content before it reaches the end user. It is designed to help make AI systems **safer, more predictable, and more trustworthy**.

### What this project does

* Applies **safety checks** to AI prompts and responses
* Flags or blocks harmful, unsafe, or policy-violating output
* Demonstrates how guardrails can be integrated into AI-powered applications
* Provides a simple frontend to visualise and test guardrail behaviour

The project is intended as a **proof of concept / learning project** for understanding how AI guardrails work and how they can be extended for real-world AI systems.
